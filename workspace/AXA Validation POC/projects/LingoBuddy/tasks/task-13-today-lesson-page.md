# Task 13：前端 — 今日课时页面

## 目标
实现今日学习任务的交互页面，支持 3 种 Task 类型。

## 范围

### 页面 (`src/views/TodayLessonView.vue`)

#### 页面结构
- 顶部：Lesson 标题 + Day N 标记 + 进度（3/5 tasks）
- 中间：Task 列表 / 单个 Task 卡片（可逐一展示或列表展示）
- 底部：今日完成统计

#### Task 类型交互

**VOCAB_CARD（单词卡片）**
- 展示：单词、音标、释义、例句
- 操作：点击"已学习"按钮 → 标记完成

**QUIZ_SINGLE（单选题）**
- 展示：题干 + 4 个选项
- 操作：选择答案 → 提交 → 显示对错反馈
- 正确：绿色 ✅ + XP 动画
- 错误：红色 ❌ + 显示正确答案

**SPELLING（拼写填空）**
- 展示：中文释义/提示 + 输入框
- 操作：输入英文 → 提交 → 判断对错（忽略大小写和首尾空格）

#### 完成所有 Task 后
- 弹窗：🎉 今日打卡成功！
- 显示获得的 XP 和 Coins
- 若升级 → 额外显示升级提示
- 按钮："返回 Dashboard"

### API 调用
- `GET /api/today` — 获取任务列表
- `POST /api/tasks/{taskId}/complete` — 完成单个任务

## 验收标准
- [ ] 3 种 Task 类型正确渲染和交互
- [ ] 完成 task 后立即刷新状态
- [ ] 全部完成后弹出打卡成功弹窗
- [ ] 已完成的 task 显示已完成标记，不可重复操作
