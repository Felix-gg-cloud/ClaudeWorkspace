## 技术栈
- Vue 3 + TypeScript + Vite 7
- Pinia (状态管理)
- Vue Router 4 (Hash 模式)
- SCSS (Dark Fantasy 设计系统)
- Web Audio API (8-bit 音效, 11 种: correct/wrong/combo/bossHit/playerHit/levelUp/victory/defeat/click/flip/coin)
- SpeechSynthesis API (TTS 发音, 支持 en-US / en-GB)
- **Phaser 3** (营地探索游戏引擎, Canvas 2D 程序化纹理)
