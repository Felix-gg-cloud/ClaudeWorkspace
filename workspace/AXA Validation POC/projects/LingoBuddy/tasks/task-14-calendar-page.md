# Task 14：前端 — 打卡日历页面

## 目标
实现按月展示打卡记录的日历页面。

## 范围

### 页面 (`src/views/CalendarView.vue`)

#### 日历组件
- 按月展示（可切换上/下月）
- 已打卡日期标记（绿色圆点或高亮背景）
- 今日日期标记（蓝色边框）
- 点击某天可展示当日详情（XP、Coins、Streak）

#### 统计面板
- 本月打卡天数
- 本月获得总 XP
- 当前连续打卡天数

### API 调用
- `GET /api/checkin/calendar?month=YYYY-MM` — 获取月度打卡数据

## 验收标准
- [ ] 日历正确渲染当月所有日期
- [ ] 打卡日期有视觉标记
- [ ] 切换月份数据正确刷新
- [ ] 统计数据准确

## 依赖
Task 06（打卡日历 API）, Task 10（前端框架）
