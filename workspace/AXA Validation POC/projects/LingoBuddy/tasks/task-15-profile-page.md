# Task 15：前端 — Profile 成就徽章页面

## 目标
展示用户个人信息、成就和徽章。

## 范围

### 页面 (`src/views/ProfileView.vue`)

#### 用户信息卡片
- 用户名 + 称号（与 Level 对应）
- 当前等级 + XP + Coins
- 学习天数统计
- 最长连续打卡记录

#### 成就列表
- 网格展示所有成就
- 已解锁：彩色图标 + 解锁日期
- 未解锁：灰色图标 + 解锁条件说明
- 已解锁数 / 总数

#### 徽章展示
- 与等级/成就关联的特殊徽章
- 可用 emoji 或 SVG 图标

### API 调用
- `GET /api/me` — 用户信息
- `GET /api/achievements` — 成就列表

## 验收标准
- [ ] 用户信息正确展示
- [ ] 已解锁/未解锁成就视觉区分明显
- [ ] 页面布局美观，响应式适配

## 依赖
Task 08（成就 API）, Task 10（前端框架）
