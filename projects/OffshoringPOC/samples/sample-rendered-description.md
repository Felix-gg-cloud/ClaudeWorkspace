## How the Report Is Shared

1. Power Automate saves the Excel (.xlsx) and/or PDF to SharePoint.
2. A **Teams notification** is posted to the designated channel:
   > ✅ **Offshoring DQ Report Ready** — Period: 202501  
   > Pass Rate: 58.3% | Errors: 7 | Warnings: 1  
   > 📎 [Open Excel Report](https://sharepoint_link) | [Open PDF](https://sharepoint_pdf_link)
3. An **email** is sent to the requestor with the same links.
4. The files remain in SharePoint for audit/archive purposes.
