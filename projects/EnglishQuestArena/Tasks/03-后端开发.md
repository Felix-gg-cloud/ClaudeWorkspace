## 核心职责（后端独占）

> 以下逻辑**只在后端实现**，前端不做判断，只负责展示后端返回的结果。

| 职责 | 说明 | 前端对应 |
|------|------|---------|
| **判卷** | 比对答案是否正确，返回 correct/wrong + explanation | 前端提交答案 → 显示结果 |
| **XP/金币计算** | 根据答题结果+连击+技能加成计算奖励 | 前端展示后端返回的奖励数值 |
| **等级升级** | XP 达到阈值自动升级，返回新等级+技能点 | 前端播放升级动画 |
| **章节状态流转** | locked→camp→quest→boss→completed | 前端查询当前状态, 渲染对应页面 |
| **SRS 调度** | SM-2 算法计算下次复习时间 | 前端只显示后端推送的复习题 |
| **Auto-Drill 出题** | seed 题不足时自动生成 MCQ/听选/拼写题 | 前端无感知, 当普通题渲染 |
| **Boss 抽题** | 按抽题计划从题池随机选题 | 前端只渲染后端返回的题目 |
| **打卡记录** | 每日完成首课自动打卡, 计算连续天数 | 前端展示打卡日历 |
| **每日限制** | Boss 每日 1 次, CEFR 测验每日限次 | 前端提示剩余次数 |

## SRS SM-2 算法伪代码

```
function updateSRS(record, quality):
    // quality: 0=完全忘记, 3=勉强记住, 5=完全掌握
    if quality >= 3:  // 正确
        if record.repetitions == 0:
            record.interval = 1
        elif record.repetitions == 1:
            record.interval = 6
        else:
            record.interval = round(record.interval * record.easeFactor)
        record.repetitions += 1
    else:  // 错误
        record.repetitions = 0
        record.interval = 1
    
    record.easeFactor = max(1.3,
        record.easeFactor + (0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02)))
    record.nextReviewDate = today + record.interval days
```

## Auto-Drill 自动出题规则

1. 取 seed tasks (按 orderIndex 排序)
2. 若 tasks.size < targetTaskCount (默认 30) 且 autoDrillEnabled=true:
   - 收集本 lesson 关联的所有 contentItems (去重)
   - 只选 type=WORD 的 contentItems
   - 按比例生成:
     - 50%: MCQ (词义选择: "Choose the meaning of 'word'")
     - 30%: MCQ (听音选词: ttsTextEn=word, 3-4 个同章节词作选项)
     - 20%: SPELLING (输入拼写: ttsTextEn=word)
3. Auto tasks code 以 `AUTO_` 前缀, xpReward=2~4, goldReward=0
4. Auto tasks 不写入 Task 表, 运行时生成; 但答题结果写入 task_progress
