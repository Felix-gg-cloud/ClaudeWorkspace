# Task 09：后端单元测试

## 目标
编写 PRD 要求的 3 个核心单元测试 + 额外补充测试。

## 范围

### 必须测试（PRD 要求）
1. **Level 计算测试**
   - 给定 totalXp = 0 → Level 1
   - 给定 totalXp = 500 → Level 2
   - 给定 totalXp = 1500 → Level 3
   - 边界值测试

2. **Streak 计算测试**
   - 连续 3 天打卡 → streak = 3
   - 中间断 1 天 → streak 归零重新计算
   - 今天首次打卡 → streak = 1

3. **Checkin 幂等测试**
   - 同日调用两次打卡 → 数据库只有 1 条记录
   - 第二次调用不报错，返回已有记录

### 补充测试（建议）
4. TaskProgress 幂等 — 重复完成同一 task 不重复计 XP
5. Seed 导入幂等 — 重复执行不报错不重复插入

## 验收标准
- [ ] `mvn test` 全部通过
- [ ] 覆盖 3 个核心场景 + 至少 1 个补充
